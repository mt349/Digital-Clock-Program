# Digital Clock Program ⏰

A simple digital clock built using HTML, CSS, and JavaScript.  
It displays the current time in real-time and updates every second.

---

## 🚀 Features

- Live digital clock display
- 12-hour format with AM/PM
- Updates every second automatically
- Clean centered UI
- Background image with modern glass-like clock design

---

## 🛠️ Technologies Used

- HTML
- CSS
- JavaScript (Date API, setInterval)

---

## 📂 Project Structure
/project-folder
│── index.html
│── style.css
│── index.js
│── nyc.image.jpg


---

## ⚙️ How It Works

- JavaScript gets the current time using `Date()`
- Hours, minutes, and seconds are extracted
- Time is formatted into 12-hour format
- The DOM is updated every 1000ms (1 second) using `setInterval()`

---

## ▶️ How to Run

1. Download or clone the project
2. Open `index.html` in your browser
3. The clock will start automatically
